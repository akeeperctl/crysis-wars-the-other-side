Доброго время суток! Это большое обновление мода если сравнить то, что было 1 год назад и сейчас, но пока что в демо-версии. Почему демо-версия? Потому-что мод буквально вгрызается в код игры, затрагивая каждый её аспект, поэтому контролировать всё и вся становится всё труднее. Но не смотря на это работа не стоит на месте и мы потихоньку двигаемся к релизу.

В этой версии мода многое стало по новому. Новая система управления, использующая архетипы врагов, нормальная камера, не уезжающая сквозь объекты и т.д.
Вас ждёт 2 уровня, один из которых вы уже видели если играли в старую версию. Оба уровня были немного обрезаны для демо версии, всё же не стоит раскрывать все карты сразу.

А теперь немного о консольных командах, доступных без режима разработчика:
ctrl_AlienTargetx
ctrl_AlienTargety 
ctrl_AlienTargetz 
ctrl_alFov

ctrl_TrooperTargetx
ctrl_TrooperTargety
ctrl_TrooperTargetz
ctrl_trFov

ctrl_ScoutTargetx
ctrl_ScoutTargety 
ctrl_ScoutTargetz 
ctrl_scFov

ctrl_HunterTargetx
ctrl_HunterTargety
ctrl_HunterTargetz
ctrl_htFov

Команды, пишущиеся вот так "ctrl_HunterTargetx" регулируют положение камеры, для каждого отдельно взятого архетипа. На конце видно "x" - это координата, по которой изменяется положение камеры.
А, команды такие как эта "ctrl_htFov" регулируют FOV.

Про установку:
Архив распаковать в папку с игрой, пример: C:\Games\Crysis Wars\ - папка с игрой, сюда и распаковать.

Про смену языка:
Найти "Default.lng" в Crysis Wars\Game\Localized\ либо создать. Открыть текстовым ред. и прописать/изменить потом сохранить:

Language = "English" - язык Russian/English
PAK = "English.pak" - архив Russian.pak/English.pak

Good day! This is a big update of the mod if you compare what was 1 year ago and now, but so far in the demo version. Why demo version? Because the mod literally bites into the code of the game, affecting every aspect of it, so it becomes more and more difficult to control everything and everyone. But despite this, the work does not stand still and we are slowly moving towards the release.

In this version of the mod, a lot has changed. A new control system that uses enemy archetypes, a normal camera that does not leave through objects, etc.

There are 2 levels waiting for you, one of which you have already seen if you played the old version. Both levels were slightly cut for the demo version, but you shouldn't reveal all the cards at once.

And now a little about the console commands available without developer mode:
ctrl_AlienTargetx
ctrl_AlienTargety
ctrl_AlienTargetz
ctrl_alFov

ctrl_TrooperTargetx
ctrl_TrooperTargety
ctrl_TrooperTargetz
ctrl_trFov

ctrl_ScoutTargetx
ctrl_ScoutTargety
ctrl_ScoutTargetz
ctrl_scFov

ctrl_HunterTargetx
ctrl_HunterTargety
ctrl_HunterTargetz
ctrl_htFov

Commands written like this "ctrl_HunterTargetx" adjust the position of the camera, for each individual archetype. At the end you can see "x" - this is the coordinate along which the position of the camera changes.
And, commands like this "ctrl_htFov" adjust the FOV.

About installation:
Unpack the archive to the game folder, for example: C:\Games\Crysis Wars\ - the game folder, unpack here.

About language change:
Find "Default.lng" in Crysis Wars\Game\Localized\ or create one. Open text ed. and register / change then save:

Language = "English" - language Russian / English
PAK = "English.pak" - archive Russian.pak / English.pak
